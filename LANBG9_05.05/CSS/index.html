A CSS angolul cascading style sheets a számítástechnikában egy stílusleíró nyelv, mely a HTML egy XHTML típusú struktúrált dokumentumok megjelenését írja le.


A CSS elnevezésénél a Cascade szó rangsorolására utal, tehát nem mindegy, hogy az adott stílus hol helyezkedik el a rangsorban. Az ütközések elkerülése végett még pár egyszerű szabály létezik. Az öröklés az YHTML elemek egymásba ágyazottságán alapul, így az egymásba ágyazott elemek öröklik az őket tartalmazó elem beállításait. A szűkítésnek a stílusok súlyozásánál van szerepe. Azaz ha meg akarjuk kölünböztetni egy stílus fontosságát, akkor szükebben, vagyis pontosabban kell megadnunk azt, így vehetjük rá a böngészőt, hogy használja. A stílusok elhelyezkedésüktől függően fontossági sorrendben állnak: Böngészőstílus, Felhasználói stílus, Beágyazott stílus, Beágyazott stílus, Szövegközi stílus
